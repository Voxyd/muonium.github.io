Technical

Coding rules

  • All the project's servers must run on BSD or Linux.
  • Only free softwares can be installed on project's servers.
  • All project's must be coded from zero by ourselves, excepted for encryption library.

Security

  • Passwords must be communicated in email on ProtonMail mail servers
  • Sensible/Private datum of Kantum Technologies or its projects must be sent in email on ProtonMail

Ethical

Money

You can make a donation for the servers, the team/enterprise, or for the conferences. We are not building this project to make money, but to make a Internet which respects users privacy.

Technical

Crytography

  • The key length must be free

Opensource

  • All projects must be published in :
    • LGPL version 3
    • Or GPL version 3
    • Or Apache License version 2
  • Libraries must be published in Apache License version 2.0

Privacy

User's privacy

  • No advertisement authorized
  • Datum must be encrypted at the client side

Government

  • We are not allowed to give datum to any government
  • If we must do it, we have to contact the user(s) concerned before

Internal social system

Democracy

  • All decisions must be approved by vote
  • All members can vote
  • For doubt decisions, we can propose referendum to have the opinion of the users
  • Any member can propose a referendum at any moment

Trust

We don't want your trust. We want to build a project to give a privacy to everybody. We are not making a commercial project, but we would like to live by it. In fact, we would like to dedicate our time to this project.

Contact

If you think that there's a misunderstanding in a rule, do not hesisate to contact us

Even if you have a comment, don't hesitate please.
